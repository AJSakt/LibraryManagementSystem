Library Management System

 Project Overview
A desktop-based Library Management System developed using Java Swing and MySQL.  
This system helps manage books, students, and book issue operations efficiently.

Technologies Used
- Java
- Java Swing (GUI)
- JDBC
- MySQL
- NetBeans IDE

Features
- Admin Login
- Add / Update / Delete Books
- Add / Update / Delete Students
- Issue Book Management
- Dashboard Panel
- Database Connectivity

Project Structure
- BookForm.java
- StudentForm.java
- IssueForm.java
- LoginForm.java
- Dashboard.java
- DBConnection.java

Future Improvements
- Return book functionality
- Fine calculation
- Search feature
- Reports generation
